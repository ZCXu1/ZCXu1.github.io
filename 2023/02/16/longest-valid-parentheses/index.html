LeetCode Top 100 - Longest Valid Parentheses

发表于 2023-02-16 分类于 LeetCode Top 100

LeetCode Page

Given a string containing just the characters '(' and ')', return the length of the longest valid (well-formed) parentheses substring.

Example:

Solution

Analysis

这种题一般都往动态规划上去考虑，设一个dp数组，dp[i]表示以i结尾的所求字符串的长度，那么显然，如果s[i]=='('，则dp[i]=0，所以只需要考虑s[i]==')'，则s[i-1] == '(' or ')'，那么就分情况考虑，若s[i-1] == '('，即".......()"样式的字符串，则dp[i] = dp[i-2]+2，若s[i-1] == ')'，即"......))"样式的字符串，假设往前dp[i-1]个长度，再前面字符是x，也就是"....x(......))"，如果s[i - dp[i-1] - 1] == x == '('，也就是"...((.....))"，则这段长度len = dp[i-1] + 2，但是小心，闭合之后前面可能还有合法的字符串，"....(...)((......))"，所以实际上长度应该是dp[i-1] + 2 + dp[i-dp[i-1]-2]，如果x == ')'，那么显然根据dp[i-1]的定义，s[i] 是多余的，意思是x都找不到闭合的左括号，更别说s[i]了，所以dp[i] = 0。
