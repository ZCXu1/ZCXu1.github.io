LeetCode Page

Given an integer array nums and an integer k, return the k most frequent elements. You may return the answer in any order.

Solution

Analysis

Top K问题是一个经典的问题，用堆(Heap)去做。比如我们要求出一个数组中最大的K个数，就可以建立一个大小为K的小顶堆，小顶堆是为了快速拿出小数排除掉。遍历数组，往堆里面添加数组的元素，如果数组大小大于K了，直接拿走堆顶，遍历数组完成后，堆里也就是数组中最大的K个数。

但是这题是出现次数的TOP K问题，其实是一样的，首先用Map统计出次数来，然后堆按次数排序去建立即可，可以写一个类来存储，Queue<Integer> heap = new PriorityQueue<>((n1, n2) -> count.get(n1) - count.get(n2));更巧妙。
