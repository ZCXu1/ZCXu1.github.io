LeetCode Top 100 - Longest Increasing Subsequence

发表于 2023-02-17 分类于 LeetCode Top 100

LeetCode Page

Given an integer array nums, return the length of the longest strictly increasing subsequence.

Example:

Solution

Analysis

这也是动态规划题目的一个经典例子，最长递增子序列，设一个dp数组，dp[i]表示以i结尾的[0...i]数组中的最长递增子序列的长度，那么对于i+1，我们只需考察[0...i]的每一个元素，和nums[i]比大小，如果比它小，那么显然dp[i+1] = dp[j] + 1, 0<= j <=i，我们取这一轮循环中最大的dp[i+1]即可。
