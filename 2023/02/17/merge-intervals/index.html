LeetCode Top 100 - Merge Intervals

发表于 2023-02-17 分类于 LeetCode Top 100

LeetCode Page

Given an array of intervals where intervals[i] = [starti, endi], merge all overlapping intervals, and return an array of the non-overlapping intervals that cover all the intervals in the input.

Example

Solution

Analysis

核心思路是先按照区间的左边界排序，排序后方便合并。假设相邻的是两个区间[a,b]和[c,d]，a<=c，那么我们只需要比较b和c与d的关系即可。[0,2],[3,4]，先比较b和c，再比较b和d，[0,2], [0,3] -> [0,3]，[0,2], [1,3] -> [0,3]
